## 🛠️ Installation

1. **Prerequisites:** Ensure you have [Fabric](https://fabricmc.net/use/installer) & the [Fabric API](https://fabricmc.net) installed
2. **Download:** Go to the [Releases](https://github.com/Moid-M/Moid-Client/releases) tab and download the latest `.jar` file.
3. **Move:** Drop the `.jar` into your Minecraft `mods` folder.
4. **Launch:** Start Minecraft and enjoy!

> [!NOTE]
> ℹ️ Info: my Meteor Client addon [Moid-Addon](https://github.com/Moid-M/Moid-Addon) is **preinstalled** if you do not want it inside your client and raw you will have to clone the repo, remove the modules manually and compile it yourself
>



## 🧩 The Full Suite

- **Moid-Addon included:** The [Moid-Addon](https://github.com/Moid-M/Moid-Addon) is pre installed with the Client
- **Meteor Addon compatible:** This Client allows you to run Meteor-Client Addons and use them without having it installed
- **Customization:** allows you to customize the client GUI, backround, colors and launch menu
- **Homescreen:** customizable home screen using animations like the liquidbounce client has
